## Discord OAuth2 Examples
These are some Discord Oauth2 implementation examples i made for some JavaScript web server libraries because why not ;)

### Format
- Each example has two files; a CommonJS and ES Module format file.
- The name of the file is the name of the library it's for. Eg: hono.cjs for Hono, or express.mjs for Express.js
